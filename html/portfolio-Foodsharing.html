Gamified Peer-to-Peer Foodsharing App [Bachelor Thesis]

2023 | Software Development & UI/UX

This mobile application was developed as the practical component of my Bachelor thesis. It focuses on combining gamification elements with a functional peer-to-peer platform to reduce food waste.

Technical Stack: C#, Unity Engine, Real-time Networking (Photon/PUN2), Android SDK

Role: Lead Developer (Software Architecture, Networking Logic, UI/UX)

Technical Highlights:
  • Software Architecture: Designed a modular C# system to handle user data, rewards, and item states efficiently.
  • Networking & Synchronization: Implemented real-time data synchronization for peer-to-peer interactions, focusing on low-latency updates and state consistency.
  • UI/UX Framework: Developed a scalable UI system in Unity, including a custom style guide for a consistent user experience.
  • Source Code: Managed the project via Git, focusing on clean separation of concerns between game logic and networking layers.
